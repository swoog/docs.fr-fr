 Pour plus d’informations sur les méthodes et propriétés de la `My.Computer.Audio` d’objets, consultez <xref:Microsoft.VisualBasic.Devices.Audio>.  
  
 Pour plus d’informations, consultez [jouer des sons](../../../visual-basic/developing-apps/programming/computer-resources/playing-sounds.md).

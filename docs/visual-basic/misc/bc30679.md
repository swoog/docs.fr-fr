# <a name="array-initializers-are-valid-only-for-arrays-but-the-type-of-ltvariablenamegt-is-lttypenamegt"></a>Initialiseurs de tableau ne sont valides uniquement pour les tableaux, mais le type de '&lt;variablename&gt;'is'&lt;typename&gt;'
Un utilisateur a tenté d’initialiser une variable non tableau avec une liste de valeurs.  
  
 **ID d’erreur :** BC30679  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déclarez, puis initialisez la variable en tant que tableau. Par exemple :  
  
     `Dim intarray As Integer() = {1, 5, 9}`  
  
-   Initialisez la variable en tant que valeur unique. Par exemple :  
  
     `Dim intvalue As Integer = 1`

## <a name="to-correct-this-error"></a><span data-ttu-id="67cd4-111">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="67cd4-111">To correct this error</span></span>  
  
-   <span data-ttu-id="67cd4-112">Si le tableau de paramètres ne spécifie pas le type de données à étendre, ajoutez un nouveau premier paramètre qui spécifie ce type.</span><span class="sxs-lookup"><span data-stu-id="67cd4-112">If the parameter array does not specify the data type you want to extend, add a new first parameter that specifies this type.</span></span>  
  
    ```  
    <Extension()>  
    Public Sub AddTo(ByRef str As String, ByVal ParamArray addOns() As String)  
    ' Concatenate the strings in addOns to str.  
    End Sub  
    ```  
  
-   <span data-ttu-id="67cd4-113">Si le tableau de paramètres ne spécifie pas le type de données à étendre, envisagez de le modifier en tableau standard, ce qui exige un argument, au lieu d’un tableau de paramètres.</span><span class="sxs-lookup"><span data-stu-id="67cd4-113">If the parameter array does specify the data type you want to extend, consider changing it to a regular array, requiring an argument, instead of a parameter array.</span></span> <span data-ttu-id="67cd4-114">Les tableaux standard peuvent être étendus.</span><span class="sxs-lookup"><span data-stu-id="67cd4-114">Regular arrays can be extended.</span></span>  
  
    ```  
    <Extension()>  
    Public Function Sum(ByVal ints() As Integer) As Integer  
        Dim total As Integer = 0  
        For Each i As Integer In ints  
            total = total + i  
        Next i  
        Return total  
    End Function  
    ```

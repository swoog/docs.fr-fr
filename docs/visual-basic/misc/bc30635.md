# <a name="readonly-and-writeonly-cannot-be-combined"></a>Impossible de combiner 'ReadOnly' et 'WriteOnly'
Les modificateurs `ReadOnly` et `WriteOnly` s’excluent mutuellement.  
  
 **ID d’erreur :** BC30635  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Spécifiez `ReadOnly` ou `WriteOnly`, mais pas les deux.

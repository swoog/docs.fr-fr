# <a name="cannot-inherit-interface-interfacename1-because-it-could-be-identical-to-interface-interfacename2-from-which-the-interface-interfacename3-inherits-for-some-type-arguments"></a><span data-ttu-id="fd2c6-102">Ne peut pas hériter de l’interface '\<nom_interface1 >', car il peut être identique à l’interface '\<nom_interface2 >' à partir de laquelle l’interface '\<nom_interface3 >' hérite pour certains arguments de type</span><span class="sxs-lookup"><span data-stu-id="fd2c6-102">Cannot inherit interface '\<interfacename1>' because it could be identical to interface '\<interfacename2>' from which the interface '\<interfacename3>' inherits for some type arguments</span></span>
<span data-ttu-id="fd2c6-103">Une interface générique hérite d’au moins deux interfaces génériques, et deux des héritages peuvent entrer en conflit pour certaines valeurs d’arguments de type.</span><span class="sxs-lookup"><span data-stu-id="fd2c6-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="fd2c6-104">Les instructions suivantes peuvent générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="fd2c6-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
    Inherits interfaceA(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="fd2c6-105">Si `derivedInterface` est construit ou implémenté en fournissant le même type à `t1` et à `t2`, il doit hériter de deux versions de `interfaceA` avec des arguments de type identiques.</span><span class="sxs-lookup"><span data-stu-id="fd2c6-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="fd2c6-106">Il en résulte une ambiguïté quant à la version à laquelle accéder.</span><span class="sxs-lookup"><span data-stu-id="fd2c6-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="fd2c6-107">**ID d’erreur :** BC32123</span><span class="sxs-lookup"><span data-stu-id="fd2c6-107">**Error ID:** BC32123</span></span>

# <a name="end-interface-must-be-preceded-by-a-matching-interface"></a>'End Interface' doit être précédé d’un 'Interface' correspondant.
Une instruction `End Interface` se produit sans instruction `Interface` correspondante. `End Interface` doit être précédée d’une instruction `Interface` correspondante.  
  
 **ID d’erreur :** BC30252  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Vérifiez si une instruction `Interface` précédente est mal orthographiée ou non valide.  
  
2. Vérifiez que les membres `Interface` sont correctement mis en forme.

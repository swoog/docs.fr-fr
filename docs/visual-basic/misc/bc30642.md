# <a name="optional-and-paramarray-cannot-be-combined"></a>Impossible de combiner 'Optional' et 'ParamArray'
Les modificateurs `Optional` et `ParamArray` s’excluent mutuellement.  
  
 **ID d’erreur :** BC30642  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Spécifiez `ParamArray` ou `Optional`, mais pas les deux.

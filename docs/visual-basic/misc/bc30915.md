# <a name="name-cannot-expose-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-outside-the-project-through-specifier-type"></a>«\<nom >' ne peut pas exposer le type délégué sous-jacent '\<type_délégué >' de l’événement qu’il implémente en dehors du projet via \<spécificateur > '\<type > »
Le code expose un type privé en dehors de la classe publique.  
  
 **ID d’erreur :** BC30915  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Déclarez le type `Public`.  
  
     ou  
  
-   Utilisez un autre type.

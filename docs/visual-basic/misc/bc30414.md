# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-the-array-types-have-different-numbers-of-dimensions"></a>Valeur de type '\<type1 >' ne peut pas être convertie en '\<type2 >', car les types tableau ont des nombres de dimensions différents
Vous avez essayé de convertir un tableau en un tableau avec un nombre de dimensions différent.  
  
 **ID d’erreur :** BC30414  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Modifiez l’un des tableaux ou les deux pour que le nombre de dimensions corresponde.

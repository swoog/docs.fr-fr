# <a name="internal-compiler-error-code-generator-received-malformed-input"></a>Erreur interne du compilateur : le générateur de code a reçu une entrée incorrecte
Le compilateur Visual Basic ne peut pas interpréter l’un de ses fichiers écrits en interne.  
  
 **ID d’erreur :** BC30011  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Recompilez le programme pour voir si l'erreur se produit à nouveau.  
  
2.  Si l’erreur se reproduit, réinstallez le compilateur Visual Basic.  
  
3.  Si l’erreur persiste après la réinstallation, rassemblez des informations sur le contexte dans lequel est survenue l’erreur, puis avertissez les Services de support technique Microsoft.

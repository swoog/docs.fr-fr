# <a name="anonymous-type-member-or-property-propertyname-is-already-declared"></a><span data-ttu-id="d35be-102">Membre de type anonyme ou la propriété '\<nom_propriété >' est déjà déclaré</span><span class="sxs-lookup"><span data-stu-id="d35be-102">Anonymous type member or property '\<propertyname>' is already declared</span></span>
<span data-ttu-id="d35be-103">Un nom de propriété ne peut être établi qu’une seule fois dans la déclaration d’un type anonyme.</span><span class="sxs-lookup"><span data-stu-id="d35be-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="d35be-104">Par exemple, les déclarations suivantes ne sont pas valides :</span><span class="sxs-lookup"><span data-stu-id="d35be-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="d35be-105">**ID d’erreur :** BC36547</span><span class="sxs-lookup"><span data-stu-id="d35be-105">**Error ID:** BC36547</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="d35be-106">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="d35be-106">To correct this error</span></span>  
  
-   <span data-ttu-id="d35be-107">Utilisez un nom différent pour l’une des propriétés.</span><span class="sxs-lookup"><span data-stu-id="d35be-107">Choose a different name for one of the properties.</span></span>  
  
```vb  
' Valid.  
Dim anonType3 = New With {.Name = "Debra Garcia", .Label = .Name & ", President"}  
```  
  
-   <span data-ttu-id="d35be-108">Attribuez de nouveaux noms aux variables et aux propriétés à partir desquelles vous déduisez des noms et des valeurs.</span><span class="sxs-lookup"><span data-stu-id="d35be-108">Provide new names for the variables or property names from which you are inferring names and values.</span></span>  
  
```vb  
' Valid.  
Dim anonType4 = New With {Key .ProductName = product.Name, Key .CarName = car1.Name}  
```

# <a name="type-parameter-not-allowed-in-implements-clause"></a><span data-ttu-id="104a2-102">Le paramètre de type n’est pas autorisé dans la clause ’Implements’</span><span class="sxs-lookup"><span data-stu-id="104a2-102">Type parameter not allowed in 'Implements' clause</span></span>
<span data-ttu-id="104a2-103">Une clause `Implements` dans un type générique spécifie un paramètre de type comme le membre à implémenter.</span><span class="sxs-lookup"><span data-stu-id="104a2-103">An `Implements` clause in a generic type specifies a type parameter as the member to be implemented.</span></span>  
  
 <span data-ttu-id="104a2-104">Une clause `Implements` doit spécifier une interface et un membre.</span><span class="sxs-lookup"><span data-stu-id="104a2-104">An `Implements` clause must specify an interface and a member.</span></span> <span data-ttu-id="104a2-105">Elle peut passer un paramètre de type à l’interface, mais elle ne peut pas le passer au membre, ni l’utiliser comme nom du membre.</span><span class="sxs-lookup"><span data-stu-id="104a2-105">It can pass a type parameter to the interface, but it cannot pass it to the member, nor use it as the name of the member.</span></span>  
  
 <span data-ttu-id="104a2-106">Les instructions suivantes peuvent générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="104a2-106">The following statements can generate this error.</span></span>  
  
```  
Class c1(Of t)  
    Implements i1(Of t)  
    Public Sub doSomething() Implements t  
End Class  
```  
  
 <span data-ttu-id="104a2-107">**ID d’erreur :** BC32056</span><span class="sxs-lookup"><span data-stu-id="104a2-107">**Error ID:** BC32056</span></span>

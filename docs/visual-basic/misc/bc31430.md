 <span data-ttu-id="bcece-104">Dans certains cas, le compilateur peut déterminer qu’un test `TypeOf`...`Is` ne peut qu’échouer, par exemple s’il n’y a aucune relation d’héritage entre deux classes.</span><span class="sxs-lookup"><span data-stu-id="bcece-104">In some cases the compiler can determine that a `TypeOf`...`Is` test can only fail, for example if there is no inheritance relationship between two classes.</span></span>

 `End If`  
  
 <span data-ttu-id="bcece-106"><xref:System.Windows.Forms.Form> et <xref:System.Array> étant des types sans aucune relation entre eux, le compilateur peut déterminer que l’expression `TypeOf`...`Is` retourne `False` pour toutes les valeurs de `refVar`.</span><span class="sxs-lookup"><span data-stu-id="bcece-106">Because <xref:System.Windows.Forms.Form> and <xref:System.Array> are totally unrelated types, the compiler can determine that the `TypeOf`...`Is` expression returns `False` for any value of `refVar`.</span></span>

# <a name="try-statements-are-not-valid-in-the-immediate-window"></a>Les instructions 'Try' ne sont pas valides dans la fenêtre Exécution
Les instructions`Try` ne peuvent être utilisées que dans le code source.  
  
 **ID d’erreur :** BC30717  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les instructions `Try` du code de débogage.

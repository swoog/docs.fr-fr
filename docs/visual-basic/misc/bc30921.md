# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-to-region-regionname"></a><span data-ttu-id="54bef-102">«\<nom_type_dérivé >' ne peut pas hériter \<type > '\<nom_type_base_construit >', car il étend l’accès du type '\<nom_type_interne >' pour \<région > '\<regionname >'</span><span class="sxs-lookup"><span data-stu-id="54bef-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' to \<region> '\<regionname>'</span></span>
<span data-ttu-id="54bef-103">Une classe ou une interface dérivée essaie d’étendre le niveau d’accès d’un type interne en l’utilisant comme argument de type dans une classe ou une interface de base.</span><span class="sxs-lookup"><span data-stu-id="54bef-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="54bef-104">Le code suivant peut générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="54bef-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="54bef-105">Le code à l’extérieur de `containingClass` n’est pas autorisé à accéder à `internalStructure`.</span><span class="sxs-lookup"><span data-stu-id="54bef-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="54bef-106">Toutefois, `derivedClass` est accessible à partir de n’importe quel code dans le même assembly.</span><span class="sxs-lookup"><span data-stu-id="54bef-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="54bef-107">Par conséquent, `derivedClass` ne peut pas hériter de `baseClass` s’il utilise `internalStructure` comme argument de type, car cela peut exposer `internalStructure` dans l’ensemble de la région de code de définition.</span><span class="sxs-lookup"><span data-stu-id="54bef-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="54bef-108">**ID d’erreur :** BC30921</span><span class="sxs-lookup"><span data-stu-id="54bef-108">**Error ID:** BC30921</span></span>

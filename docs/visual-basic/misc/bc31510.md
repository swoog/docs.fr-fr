# <a name="attribute-constant-39ltconstantnamegt39-cannot-be-the-target-of-an-assignment"></a>Constante d’attribut &#39; &lt;nom_constante&gt; &#39; ne peut pas être la cible d’une assignation
L’utilisateur a tenté d’assigner une valeur à une constante déclarée dans un attribut.  
  
 **ID d’erreur :** BC31510  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez l’assignation erronée.  
  
-   Si vous utilisez des attributs personnalisés que vous avez développés, redéfinissez le membre dans l’attribut en tant que champ.

# <a name="typename-is-a-type-in-containername-and-cannot-be-used-as-an-expression"></a>«\<nom_type >' est un type dans '\<containername >' et ne peut pas être utilisé en tant qu’expression
L’un des noms utilisés est déjà déclaré comme nom d’un type et ne peut pas être utilisé dans une expression.  
  
 **ID d’erreur :** BC30691  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Attribuez un nom unique au membre.  
  
-   Mettez le nom du membre entre crochets.

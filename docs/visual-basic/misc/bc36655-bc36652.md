# <a name="data-types-of-the-type-parameters-in-extension-method-ltmethodnamegt-defined-in-lttypenamegt-cannot-be-inferred-from-these-arguments-because-more-than-one-type-is-possible"></a><span data-ttu-id="9968a-102">Ou les types de données des paramètres de type dans la méthode d’extension '&lt;nom_méthode&gt;'définie dans'&lt;typename&gt;' ne peut pas être déduit à partir de ces arguments, car plusieurs types sont possibles</span><span class="sxs-lookup"><span data-stu-id="9968a-102">Data type(s) of the type parameter(s) in extension method '&lt;methodname&gt;' defined in '&lt;typename&gt;' cannot be inferred from these arguments because more than one type is possible</span></span>
<span data-ttu-id="9968a-103">Ou les types de données des paramètres de type dans la méthode d’extension '\<nom_méthode >' définie dans '\<nom_type >' ne peut pas être déduit à partir de ces arguments, car plusieurs types sont possibles.</span><span class="sxs-lookup"><span data-stu-id="9968a-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in '\<typename>' cannot be inferred from these arguments because more than one type is possible.</span></span> <span data-ttu-id="9968a-104">La spécification explicite du ou des types de données peut permettre de corriger cette erreur.</span><span class="sxs-lookup"><span data-stu-id="9968a-104">Specifying the data type(s) explicitly might correct this error.</span></span>  
  
 <span data-ttu-id="9968a-105">L’utilisateur a tenté d’utiliser l’inférence de type pour déterminer le ou les types du ou des paramètres de type dans un appel à une méthode d’extension générique.</span><span class="sxs-lookup"><span data-stu-id="9968a-105">An attempt has been made to use type inference to determine the type (or types) of the type parameter (or parameters) in a call to a generic extension method.</span></span> <span data-ttu-id="9968a-106">Le compilateur détecte plusieurs types de données possibles pour un ou plusieurs des paramètres de type, et il signale cette erreur.</span><span class="sxs-lookup"><span data-stu-id="9968a-106">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="9968a-107">Quand la spécification des arguments n’est pas une option (par exemple, pour les opérateurs de requête dans les expressions de requête), le message d’erreur apparaît sans la deuxième phrase.</span><span class="sxs-lookup"><span data-stu-id="9968a-107">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>  
  
 <span data-ttu-id="9968a-108">Le code suivant illustre cette erreur.</span><span class="sxs-lookup"><span data-stu-id="9968a-108">The following code demonstrates the error.</span></span>  
  
```vb  
Option Strict Off  
Imports System.Runtime.CompilerServices  
Module Module1  
    Sub Main()  
  
        Dim caller As New Class1  
        '' Not valid.  
        'caller.targetExtension(1, "2")  
  
    End Sub  
  
    <Extension()> _  
    Sub targetExtension(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)  
    End Sub  
  
    Class Class1  
    End Class  
  
End Module  
```

# <a name="continue-must-be-followed-by-do-for-or-while"></a><span data-ttu-id="75e5e-102">'Continue' doit être suivi par 'Do', 'For' ou 'While'</span><span class="sxs-lookup"><span data-stu-id="75e5e-102">'Continue' must be followed by 'Do', 'For' or 'While'</span></span>
<span data-ttu-id="75e5e-103">Une instruction `Continue` doit être suivie de `Do`, `For`ou `While`, selon que l’instruction `Continue` apparaît dans une boucle `Do...Loop` , `For...Next` ou `While...End While` .</span><span class="sxs-lookup"><span data-stu-id="75e5e-103">A `Continue` statement must be followed by `Do`, `For`, or `While`, depending on if the `Continue` statement appears within a `Do...Loop` loop, `For...Next` loop, or `While...End While` loop.</span></span>  
  
 <span data-ttu-id="75e5e-104">**ID d’erreur :** BC30781</span><span class="sxs-lookup"><span data-stu-id="75e5e-104">**Error ID:** BC30781</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="75e5e-105">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="75e5e-105">To correct this error</span></span>  
  
1. <span data-ttu-id="75e5e-106">Si l’instruction `Continue` se trouve dans une boucle `Do...Loop` , remplacez l’instruction par `Continue Do`.</span><span class="sxs-lookup"><span data-stu-id="75e5e-106">If the `Continue` statement is in a `Do...Loop` loop, change the statement to `Continue Do`.</span></span>

3. <span data-ttu-id="75e5e-108">Si l’instruction `Continue` se trouve dans une boucle `While...End While` , remplacez l’instruction par `Continue While`.</span><span class="sxs-lookup"><span data-stu-id="75e5e-108">If the `Continue` statement is in a `While...End While` loop, change the statement to `Continue While`.</span></span>  
  
4. <span data-ttu-id="75e5e-109">Sinon, supprimez l’instruction `Continue` .</span><span class="sxs-lookup"><span data-stu-id="75e5e-109">Otherwise, remove the `Continue` statement.</span></span>

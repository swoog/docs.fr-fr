# <a name="method-cannot-contain-both-a-try-statement-and-an-on-error-or-resume-statement"></a>Une méthode ne peut pas contenir à la fois une instruction 'Try' et une instruction 'On Error' ou 'Resume'
Vous avez associé une instruction `Try` à `On Error` ou `Resume`.  
  
 **ID d’erreur :** BC30544  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1.  Supprimez `On Error` ou `Resume`.

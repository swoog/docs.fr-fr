# <a name="class-in-an-interface-cannot-be-declared-ltspecifiergt"></a>Une classe dans une interface ne peut pas être déclarée '&lt;spécificateur&gt;'
Un modificateur non valide a été spécifié avec une classe déclarée dans une interface.  
  
 **ID d’erreur :** BC31070  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Supprimez les modificateurs tels que `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`ou `Overridable`.

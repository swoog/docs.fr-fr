 <span data-ttu-id="e801a-111">Les *préfixes d’espace de noms* XML constituent un autre aspect des noms XML.</span><span class="sxs-lookup"><span data-stu-id="e801a-111">Another aspect of XML names is XML *namespace prefixes*.</span></span> <span data-ttu-id="e801a-112">Les préfixes XML sont la cause de la plupart de la complexité des noms XML.</span><span class="sxs-lookup"><span data-stu-id="e801a-112">XML prefixes cause most of the complexity of XML names.</span></span> <span data-ttu-id="e801a-113">Ces préfixes vous permettent de créer un raccourci pour un espace de noms XML, ce qui rend le document XML plus concis et plus compréhensible.</span><span class="sxs-lookup"><span data-stu-id="e801a-113">These prefixes enable you to create a shortcut for an XML namespace, which makes the XML document more concise and understandable.</span></span> <span data-ttu-id="e801a-114">Toutefois, la signification des préfixes XML dépend de leur contexte, ce qui augmente la complexité.</span><span class="sxs-lookup"><span data-stu-id="e801a-114">However, XML prefixes depend on their context to have meaning, which adds complexity.</span></span> <span data-ttu-id="e801a-115">Par exemple, le préfixe XML `aw` pourrait être associé à un espace de noms XML dans une partie d'une arborescence XML et à un espace de noms XML différent dans une autre partie de l'arborescence XML.</span><span class="sxs-lookup"><span data-stu-id="e801a-115">For example, the XML prefix `aw` could be associated with one XML namespace in one part of an XML tree, and with a different XML namespace in a different part of the XML tree.</span></span>

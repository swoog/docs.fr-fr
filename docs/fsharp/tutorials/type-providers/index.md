# <a name="type-providers"></a><span data-ttu-id="da099-103">Fournisseurs de type</span><span class="sxs-lookup"><span data-stu-id="da099-103">Type Providers</span></span>

<span data-ttu-id="da099-104">Un fournisseur de type F# est un composant qui fournit des types, des propriétés et des méthodes à utiliser dans votre programme.</span><span class="sxs-lookup"><span data-stu-id="da099-104">An F# type provider is a component that provides types, properties, and methods for use in your program.</span></span> <span data-ttu-id="da099-105">Fournisseurs de type génèrent ce que l'on **Types fourni**, qui sont généré par le compilateur F # et sont basé sur une source de données externe.</span><span class="sxs-lookup"><span data-stu-id="da099-105">Type Providers generate what are known as **Provided Types**, which are generated by the F# compiler and are based on an external data source.</span></span>

<span data-ttu-id="da099-106">Par exemple, un fournisseur de Type F # pour SQL peut générer des types représentant les tables et colonnes dans une base de données relationnelle.</span><span class="sxs-lookup"><span data-stu-id="da099-106">For example, an F# Type Provider for SQL can generate types representing tables and columns in a relational database.</span></span> <span data-ttu-id="da099-107">En fait, c’est ce que le [SQLProvider](https://fsprojects.github.io/SQLProvider/) fournisseur de Type ne.</span><span class="sxs-lookup"><span data-stu-id="da099-107">In fact, this is what the [SQLProvider](https://fsprojects.github.io/SQLProvider/) Type Provider does.</span></span>

<span data-ttu-id="da099-108">Fourni que types dépendent des paramètres d’entrée à un fournisseur de Type.</span><span class="sxs-lookup"><span data-stu-id="da099-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="da099-109">Entrée de ce genre peut être un exemple de source de données (par exemple, un fichier de schéma JSON), une URL qui pointe directement vers un service externe ou une chaîne de connexion à une source de données.</span><span class="sxs-lookup"><span data-stu-id="da099-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="da099-110">Un fournisseur de Type peut également vous assurer que les groupes de types sont développés uniquement à la demande ; Autrement dit, ils sont développés si les types sont réellement référencés par votre programme.</span><span class="sxs-lookup"><span data-stu-id="da099-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="da099-111">Cela permet l'intégration directe et à la demande d'espaces d'informations à grande échelle, tels que les marchés de données en ligne, de manière fortement typée.</span><span class="sxs-lookup"><span data-stu-id="da099-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

## <a name="generative-and-erased-type-providers"></a><span data-ttu-id="da099-112">Fournisseurs de Type générative et effacé</span><span class="sxs-lookup"><span data-stu-id="da099-112">Generative and Erased Type Providers</span></span>

<span data-ttu-id="da099-113">Fournisseurs de type se présentent sous deux formes : générative et effacées.</span><span class="sxs-lookup"><span data-stu-id="da099-113">Type Providers come in two forms: Generative and Erased.</span></span>

<span data-ttu-id="da099-114">Les fournisseurs de Type générative produisent des types qui peuvent être écrits en tant que types .NET dans l’assembly dans lequel ils sont générés.</span><span class="sxs-lookup"><span data-stu-id="da099-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="da099-115">Cela leur permet d’être utilisé à partir du code dans d’autres assemblys.</span><span class="sxs-lookup"><span data-stu-id="da099-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="da099-116">Cela signifie que la représentation typée de la source de données doit être généralement un est possible de représenter avec des types .NET.</span><span class="sxs-lookup"><span data-stu-id="da099-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>

<span data-ttu-id="da099-117">Effacement des fournisseurs de Type produisent des types qui peuvent uniquement être utilisées dans l’assembly ou le projet, de qu'ils sont générés à partir.</span><span class="sxs-lookup"><span data-stu-id="da099-117">Erasing Type Providers produce types that can only be consumed in the assembly or project they are generated from.</span></span> <span data-ttu-id="da099-118">Les types sont éphémères ; Autrement dit, ils ne sont pas écrites dans un assembly et ne peut pas être consommés par le code dans d’autres assemblys.</span><span class="sxs-lookup"><span data-stu-id="da099-118">The types are ephemeral; that is, they are not written into an assembly and cannot be consumed by code in other assemblies.</span></span> <span data-ttu-id="da099-119">Elles peuvent contenir des *retardée* membres, vous permettant d’utiliser les types à partir d’un espace potentiellement infini d’informations.</span><span class="sxs-lookup"><span data-stu-id="da099-119">They can contain *delayed* members, allowing you to use provided types from a potentially infinite information space.</span></span> <span data-ttu-id="da099-120">Elles sont utiles pour l’utilisation d’un petit sous-ensemble d’une source de données volumineux et interconnectées.</span><span class="sxs-lookup"><span data-stu-id="da099-120">They are useful for using a small subset of a large and interconnected data source.</span></span>

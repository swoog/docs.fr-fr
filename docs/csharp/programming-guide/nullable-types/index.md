# <a name="nullable-types-c-programming-guide"></a><span data-ttu-id="86588-103">Types Nullable (Guide de programmation C#)</span><span class="sxs-lookup"><span data-stu-id="86588-103">Nullable types (C# Programming Guide)</span></span>

<span data-ttu-id="86588-104">Les types Nullable sont des instances du struct <xref:System.Nullable%601?displayProperty=nameWithType>.</span><span class="sxs-lookup"><span data-stu-id="86588-104">Nullable types are instances of the <xref:System.Nullable%601?displayProperty=nameWithType> struct.</span></span> <span data-ttu-id="86588-105">Les types Nullable peuvent représenter toutes les valeurs d’un type sous-jacent `T` ainsi qu’une autre valeur [null](../../language-reference/keywords/null.md).</span><span class="sxs-lookup"><span data-stu-id="86588-105">Nullable types can represent all the values of an underlying type `T`, and an additional [null](../../language-reference/keywords/null.md) value.</span></span> <span data-ttu-id="86588-106">Le type sous-jacent `T` peut être n’importe quel [type valeur](../../language-reference/keywords/value-types.md) non nullable.</span><span class="sxs-lookup"><span data-stu-id="86588-106">The underlying type `T` can be any non-nullable [value type](../../language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="86588-107">`T` ne peut pas être un type référence.</span><span class="sxs-lookup"><span data-stu-id="86588-107">`T` cannot be a reference type.</span></span>

<span data-ttu-id="86588-108">Par exemple, vous pouvez affecter `null` ou toute valeur entière à partir de <xref:System.Int32.MinValue?displayProperty=nameWithType> à <xref:System.Int32.MaxValue?displayProperty=nameWithType> à un `Nullable<int>` et [true](../../language-reference/keywords/true-literal.md), [false](../../language-reference/keywords/false-literal.md) ou `null` à un `Nullable<bool>`.</span><span class="sxs-lookup"><span data-stu-id="86588-108">For example, you can assign `null` or any integer value from <xref:System.Int32.MinValue?displayProperty=nameWithType> to <xref:System.Int32.MaxValue?displayProperty=nameWithType> to a `Nullable<int>` and [true](../../language-reference/keywords/true-literal.md), [false](../../language-reference/keywords/false-literal.md), or `null` to a `Nullable<bool>`.</span></span>

<span data-ttu-id="86588-109">Utilisez un type nullable lorsque vous avez besoin de représenter la valeur non definie d’un type sous-jacent.</span><span class="sxs-lookup"><span data-stu-id="86588-109">You use a nullable type when you need to represent the undefined value of an underlying type.</span></span> <span data-ttu-id="86588-110">Une variable booléenne peut avoir uniquement deux valeurs : true et false.</span><span class="sxs-lookup"><span data-stu-id="86588-110">A Boolean variable can have only two values: true and false.</span></span> <span data-ttu-id="86588-111">Il n’existe aucune valeur « non définie ».</span><span class="sxs-lookup"><span data-stu-id="86588-111">There is no "undefined" value.</span></span> <span data-ttu-id="86588-112">Dans de nombreuses applications de programmation, notamment dans les interactions de base de données, une valeur de variable peut être indéfinie ou manquante.</span><span class="sxs-lookup"><span data-stu-id="86588-112">In many programming applications, most notably database interactions, a variable value can be undefined or missing.</span></span> <span data-ttu-id="86588-113">Par exemple, un champ dans une base de données peut contenir les valeurs true ou false, ou aucune valeur du tout.</span><span class="sxs-lookup"><span data-stu-id="86588-113">For example, a field in a database may contain the values true or false, or it may contain no value at all.</span></span> <span data-ttu-id="86588-114">Vous utilisez un type `Nullable<bool>` dans ce cas.</span><span class="sxs-lookup"><span data-stu-id="86588-114">You use a `Nullable<bool>` type in that case.</span></span>

<span data-ttu-id="86588-115">Les types Nullable possèdent les caractéristiques suivantes :</span><span class="sxs-lookup"><span data-stu-id="86588-115">Nullable types have the following characteristics:</span></span>
  
- <span data-ttu-id="86588-116">Les types Nullable représentent des variables de type valeur qui peuvent avoir la valeur `null`.</span><span class="sxs-lookup"><span data-stu-id="86588-116">Nullable types represent value-type variables that can be assigned the `null` value.</span></span> <span data-ttu-id="86588-117">Il n’est pas possible de créer un type Nullable à partir d’un type référence.</span><span class="sxs-lookup"><span data-stu-id="86588-117">You cannot create a nullable type based on a reference type.</span></span> <span data-ttu-id="86588-118">(Les types référence prennent déjà en charge la valeur `null`.)</span><span class="sxs-lookup"><span data-stu-id="86588-118">(Reference types already support the `null` value.)</span></span>  
  
- <span data-ttu-id="86588-119">La syntaxe `T?` est l’abréviation de `Nullable<T>`.</span><span class="sxs-lookup"><span data-stu-id="86588-119">The syntax `T?` is shorthand for `Nullable<T>`.</span></span> <span data-ttu-id="86588-120">Les deux formats sont interchangeables.</span><span class="sxs-lookup"><span data-stu-id="86588-120">The two forms are interchangeable.</span></span>  
  
- <span data-ttu-id="86588-121">Affectez une valeur à un type Nullable, comme vous le feriez pour un type valeur sous-jacent : `int? x = 10;` ou `double? d = 4.108;`.</span><span class="sxs-lookup"><span data-stu-id="86588-121">Assign a value to a nullable type just as you would for an underlying value type: `int? x = 10;` or `double? d = 4.108;`.</span></span> <span data-ttu-id="86588-122">Vous pouvez également affecter la valeur `null` : `int? x = null;`.</span><span class="sxs-lookup"><span data-stu-id="86588-122">You also can assign the `null` value: `int? x = null;`.</span></span>  
  
- <span data-ttu-id="86588-123">Utilisez les propriétés <xref:System.Nullable%601.HasValue%2A?displayProperty=nameWithType> et <xref:System.Nullable%601.Value%2A?displayProperty=nameWithType> en lecture seule pour rechercher les valeurs null et récupérer la valeur, comme indiqué dans l’exemple suivant : `if (x.HasValue) y = x.Value;`</span><span class="sxs-lookup"><span data-stu-id="86588-123">Use the <xref:System.Nullable%601.HasValue%2A?displayProperty=nameWithType> and <xref:System.Nullable%601.Value%2A?displayProperty=nameWithType> readonly properties to test for null and retrieve the value, as shown in the following example: `if (x.HasValue) y = x.Value;`</span></span>  
  
  - <span data-ttu-id="86588-124">La propriété <xref:System.Nullable%601.HasValue%2A> retourne `true` si la variable contient une valeur, ou `false` si elle est `null`.</span><span class="sxs-lookup"><span data-stu-id="86588-124">The <xref:System.Nullable%601.HasValue%2A> property returns `true` if the variable contains a value, or `false` if it's `null`.</span></span>
  
  - <span data-ttu-id="86588-125">La propriété <xref:System.Nullable%601.Value%2A> retourne une valeur si <xref:System.Nullable%601.HasValue%2A> retourne `true`.</span><span class="sxs-lookup"><span data-stu-id="86588-125">The <xref:System.Nullable%601.Value%2A> property returns a value if <xref:System.Nullable%601.HasValue%2A> returns `true`.</span></span> <span data-ttu-id="86588-126">Sinon, une exception <xref:System.InvalidOperationException> est levée.</span><span class="sxs-lookup"><span data-stu-id="86588-126">Otherwise, an <xref:System.InvalidOperationException> is thrown.</span></span>  
  
- <span data-ttu-id="86588-127">Vous pouvez également utiliser les opérateurs `==` et `!=` avec un type Nullable, comme le montre l’exemple suivant : `if (x != null) y = x.Value;`.</span><span class="sxs-lookup"><span data-stu-id="86588-127">You can also use the `==` and `!=` operators with a nullable type, as shown in the following example: `if (x != null) y = x.Value;`.</span></span> <span data-ttu-id="86588-128">Si `a` et `b` sont tous les deux null, `a == b` prend la valeur `true`.</span><span class="sxs-lookup"><span data-stu-id="86588-128">If `a` and `b` are both null, `a == b` evaluates to `true`.</span></span>  

- <span data-ttu-id="86588-129">À compter de C# 7.0, vous pouvez utiliser les [critères spéciaux](../../pattern-matching.md#the-is-type-pattern-expression) pour examiner et obtenir une valeur d’un type Nullable : `if (x is int valueOfX) y = valueOfX;`.</span><span class="sxs-lookup"><span data-stu-id="86588-129">Beginning with C# 7.0, you can use [pattern matching](../../pattern-matching.md#the-is-type-pattern-expression) to both examine and get a value of a nullable type: `if (x is int valueOfX) y = valueOfX;`.</span></span>
  
- <span data-ttu-id="86588-130">La valeur par défaut de `T?` est une instance dont la propriété <xref:System.Nullable%601.HasValue%2A> retourne `false`.</span><span class="sxs-lookup"><span data-stu-id="86588-130">The default value of `T?` is an instance whose <xref:System.Nullable%601.HasValue%2A> property returns `false`.</span></span>  

- <span data-ttu-id="86588-131">Utilisez la méthode <xref:System.Nullable%601.GetValueOrDefault> pour retourner la valeur affectée, ou la valeur [par défaut](../../language-reference/keywords/default-values-table.md) du type valeur sous-jacent si la valeur du type Nullable est `null`.</span><span class="sxs-lookup"><span data-stu-id="86588-131">Use the <xref:System.Nullable%601.GetValueOrDefault> method to return either the assigned value, or the [default](../../language-reference/keywords/default-values-table.md) value of the underlying value type if the value of the nullable type is `null`.</span></span>  

- <span data-ttu-id="86588-132">Utilisez la méthode <xref:System.Nullable%601.GetValueOrDefault(%600)> pour retourner la valeur affectée, ou la valeur par défaut fournie si la valeur du type Nullable est `null`.</span><span class="sxs-lookup"><span data-stu-id="86588-132">Use the <xref:System.Nullable%601.GetValueOrDefault(%600)> method to return either the assigned value, or the provided default value if the value of the nullable type is `null`.</span></span>
  
- <span data-ttu-id="86588-133">Utilisez l’[opérateur null-coalescing](../../language-reference/operators/null-coalescing-operator.md), `??`, pour affecter une valeur à un type sous-jacent basé sur une valeur du type Nullable : `int? x = null; int y = x ?? -1;`.</span><span class="sxs-lookup"><span data-stu-id="86588-133">Use the [null-coalescing operator](../../language-reference/operators/null-coalescing-operator.md), `??`, to assign a value to an underlying type based on a value of the nullable type: `int? x = null; int y = x ?? -1;`.</span></span> <span data-ttu-id="86588-134">Dans l’exemple, dans la mesure où `x` est null, la valeur de résultat de `y` est `-1`.</span><span class="sxs-lookup"><span data-stu-id="86588-134">In the example, since `x` is null, the result value of `y` is `-1`.</span></span>

- <span data-ttu-id="86588-135">Si une conversion définie par l’utilisateur est définie entre deux types de données, la même conversion peut également être utilisée avec les versions Nullable de ces types de données.</span><span class="sxs-lookup"><span data-stu-id="86588-135">If a user-defined conversion is defined between two data types, the same conversion can also be used with the nullable versions of these data types.</span></span>
  
- <span data-ttu-id="86588-136">Les types Nullable imbriqués ne sont pas autorisés.</span><span class="sxs-lookup"><span data-stu-id="86588-136">Nested nullable types are not allowed.</span></span> <span data-ttu-id="86588-137">Il n’est pas possible de compiler la ligne suivante : `Nullable<Nullable<int>> n;`</span><span class="sxs-lookup"><span data-stu-id="86588-137">The following line doesn't compile: `Nullable<Nullable<int>> n;`</span></span>

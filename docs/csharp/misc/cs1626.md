# <a name="compiler-error-cs1626"></a><span data-ttu-id="9a41c-102">Erreur du compilateur CS1626</span><span class="sxs-lookup"><span data-stu-id="9a41c-102">Compiler Error CS1626</span></span>
<span data-ttu-id="9a41c-103">Impossible de générer une valeur dans le corps d'un bloc try avec une clause catch</span><span class="sxs-lookup"><span data-stu-id="9a41c-103">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="9a41c-104">L’instruction yield n’est pas autorisée dans un bloc try si une clause catch est associée au bloc try.</span><span class="sxs-lookup"><span data-stu-id="9a41c-104">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="9a41c-105">Pour éviter cette erreur, placez l’instruction yield en dehors du bloc try/catch/finally ou supprimez le bloc catch.</span><span class="sxs-lookup"><span data-stu-id="9a41c-105">To avoid this error, either move the yield statement out of the try/catch/finally block, or remove the catch block.</span></span>
  
 <span data-ttu-id="9a41c-106">L’exemple suivant génère l’erreur CS1626 :</span><span class="sxs-lookup"><span data-stu-id="9a41c-106">The following sample generates CS1626:</span></span>  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
      finally
      {
      
      }
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```

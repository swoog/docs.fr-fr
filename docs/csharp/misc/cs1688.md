# <a name="compiler-error-cs1688"></a><span data-ttu-id="b5524-102">Erreur du compilateur CS1688</span><span class="sxs-lookup"><span data-stu-id="b5524-102">Compiler Error CS1688</span></span>
<span data-ttu-id="b5524-103">Impossible de convertir un bloc de méthode anonyme sans une liste de paramètres en type délégué 'délégué', car il a un ou plusieurs paramètres out</span><span class="sxs-lookup"><span data-stu-id="b5524-103">Cannot convert anonymous method block without a parameter list to delegate type 'delegate' because it has one or more out parameters</span></span>  
  
 <span data-ttu-id="b5524-104">Dans la plupart des cas, le compilateur autorise l’omission des paramètres d’un bloc de méthode anonyme.</span><span class="sxs-lookup"><span data-stu-id="b5524-104">The compiler allows parameters to be omitted from an anonymous method block in most cases.</span></span> <span data-ttu-id="b5524-105">Cette erreur se produit quand le bloc de méthode anonyme ne dispose pas d’une liste de paramètres alors que le délégué dispose d’un paramètre `out` .</span><span class="sxs-lookup"><span data-stu-id="b5524-105">This error arises when the anonymous method block does not have a parameter list, but the delegate has an `out` parameter.</span></span> <span data-ttu-id="b5524-106">Le compilateur n’autorise pas cette situation, car cela reviendrait à ignorer la présence du paramètre `out` , ce qui n’est pas un comportement souhaitable.</span><span class="sxs-lookup"><span data-stu-id="b5524-106">The compiler does not allow this situation because it would need to ignore the presence of the `out` parameter, which is unlikely to be the correct behavior.</span></span>

# <a name="compiler-error-cs1528"></a><span data-ttu-id="fa8c7-102">Erreur du compilateur CS1528</span><span class="sxs-lookup"><span data-stu-id="fa8c7-102">Compiler Error CS1528</span></span>
<span data-ttu-id="fa8c7-103">; ou = attendu (impossible de spécifier des arguments de constructeur dans une déclaration)</span><span class="sxs-lookup"><span data-stu-id="fa8c7-103">Expected ; or = (cannot specify constructor arguments in declaration)</span></span>  
  
 <span data-ttu-id="fa8c7-104">Une référence à une classe a été spécifiée comme si un objet de la classe était créé.</span><span class="sxs-lookup"><span data-stu-id="fa8c7-104">A reference to a class was formed as if an object to the class was being created.</span></span> <span data-ttu-id="fa8c7-105">Par exemple, une tentative de passer une variable à un constructeur a eu lieu.</span><span class="sxs-lookup"><span data-stu-id="fa8c7-105">For example, there was an attempt to pass a variable to a constructor.</span></span> <span data-ttu-id="fa8c7-106">Utilisez l’opérateur [new](../../csharp/language-reference/keywords/new.md) pour créer un objet d’une classe.</span><span class="sxs-lookup"><span data-stu-id="fa8c7-106">Use the [new](../../csharp/language-reference/keywords/new.md) operator to create an object of a class.</span></span>  
  
 <span data-ttu-id="fa8c7-107">L’exemple suivant génère l’erreur CS1528 :</span><span class="sxs-lookup"><span data-stu-id="fa8c7-107">The following sample generates CS1528:</span></span>  
  
```csharp  
// CS1528.cs  
using System;  
  
public class B  
{  
   public B(int i)  
   {  
      _i = i;  
   }  
  
   public void PrintB()  
   {  
      Console.WriteLine(_i);  
   }  
  
   private int _i;  
}  
  
public class mine  
{  
   public static void Main()  
   {  
      B b(3);   // CS1528, reference is not an object  
      // try one of the following  
      // B b;  
      // or  
      // B bb = new B(3);  
      // bb.PrintB();  
   }  
}  
```

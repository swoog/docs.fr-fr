 <span data-ttu-id="55edc-107">L’exemple suivant génère l’erreur CS0136 :</span><span class="sxs-lookup"><span data-stu-id="55edc-107">The following sample generates CS0136:</span></span>  
  
```csharp

   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;

            char i = 'a';   // CS0136, hides int i

         i++;  
      }  
   }
```  

# <a name="compiler-warning-level-1-cs3027"></a><span data-ttu-id="33edf-102">Avertissement du compilateur (niveau 1) CS3027</span><span class="sxs-lookup"><span data-stu-id="33edf-102">Compiler Warning (level 1) CS3027</span></span>
<span data-ttu-id="33edf-103">'type_1' n’est pas conforme CLS, car l’interface de base 'type_2' n’est pas conforme CLS</span><span class="sxs-lookup"><span data-stu-id="33edf-103">'type_1' is not CLS-compliant because base interface 'type_2' is not CLS-compliant</span></span>  
  
 <span data-ttu-id="33edf-104">Un type non conforme CLS ne peut pas être un type de base pour un type conforme CLS.</span><span class="sxs-lookup"><span data-stu-id="33edf-104">A non-CLS compliant type cannot be a base type for a type that is CLS compliant.</span></span>  
  
## <a name="example"></a><span data-ttu-id="33edf-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="33edf-105">Example</span></span>  
 <span data-ttu-id="33edf-106">L’exemple suivant contient une interface avec une méthode qui utilise un type non conforme CLS dans sa signature, ce qui rend le type non conforme CLS.</span><span class="sxs-lookup"><span data-stu-id="33edf-106">The following sample contains an interface with a method that uses a non-CLS compliant type in its signature, making the type non-CLS compliant.</span></span>  
  
```csharp  
// CS3027.cs  
// compile with: /target:library  
public interface IBase  
{  
   void IMethod(uint i);  
}  
```

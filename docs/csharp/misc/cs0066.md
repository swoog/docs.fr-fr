# <a name="compiler-error-cs0066"></a><span data-ttu-id="b900f-102">Erreur du compilateur CS0066</span><span class="sxs-lookup"><span data-stu-id="b900f-102">Compiler Error CS0066</span></span>
<span data-ttu-id="b900f-103">'event' : l’événement doit être de type délégué</span><span class="sxs-lookup"><span data-stu-id="b900f-103">'event': event must be of a delegate type</span></span>  
  
 <span data-ttu-id="b900f-104">Le mot clé de l’événement nécessite un type [délégué](../../csharp/language-reference/keywords/delegate.md) .</span><span class="sxs-lookup"><span data-stu-id="b900f-104">The event keyword requires a [delegate](../../csharp/language-reference/keywords/delegate.md) type.</span></span> <span data-ttu-id="b900f-105">Pour plus d’informations, consultez [événements](../../csharp/programming-guide/events/index.md) et [délégués](../../csharp/programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="b900f-105">For more information, see [Events](../../csharp/programming-guide/events/index.md) and [Delegates](../../csharp/programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="b900f-106">L’exemple suivant génère l’erreur CS0066 :</span><span class="sxs-lookup"><span data-stu-id="b900f-106">The following sample generates CS0066:</span></span>  
  
```csharp  
// CS0066.cs  
using System;  
  
public class EventHandler  
{  
}  
  
// to fix the error, remove the event declaration and the  
// EventHandler class declaration, and uncomment the following line  
// public delegate void EventHandler();  
  
public class a  
{  
   public event EventHandler Click;   // CS0066  
  
   private void TestMethod()  
   {  
      if (Click != null)  
         Click();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```

<span data-ttu-id="48c61-103">L’étiquette 'étiquette' figure déjà dans cette instruction switch</span><span class="sxs-lookup"><span data-stu-id="48c61-103">The label 'label' already occurs in this switch statement</span></span>  
  
 <span data-ttu-id="48c61-104">Une étiquette a été répétée dans une instruction [switch](../../csharp/language-reference/keywords/switch.md) .</span><span class="sxs-lookup"><span data-stu-id="48c61-104">A label was repeated in a [switch](../../csharp/language-reference/keywords/switch.md) statement.</span></span> <span data-ttu-id="48c61-105">Pour plus d'informations, consultez [switch](../../csharp/language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="48c61-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="48c61-106">L’exemple suivant génère l’erreur CS0152 :</span><span class="sxs-lookup"><span data-stu-id="48c61-106">The following sample generates CS0152:</span></span>  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```

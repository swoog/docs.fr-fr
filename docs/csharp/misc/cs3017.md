## <a name="example"></a><span data-ttu-id="58aad-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="58aad-108">Example</span></span>  
 <span data-ttu-id="58aad-109">L’exemple suivant génère l’avertissement CS3017 :</span><span class="sxs-lookup"><span data-stu-id="58aad-109">The following example generates CS3017:</span></span>

# <a name="compiler-error-cs0695"></a><span data-ttu-id="babe6-102">Erreur du compilateur CS0695</span><span class="sxs-lookup"><span data-stu-id="babe6-102">Compiler Error CS0695</span></span>
<span data-ttu-id="babe6-103">'generic type' ne peut pas implémenter 'generic interface' et 'generic interface', car ils peuvent être réunis pour des substitutions de paramètre de type</span><span class="sxs-lookup"><span data-stu-id="babe6-103">'generic type' cannot implement both 'generic interface' and 'generic interface' because they may unify for some type parameter substitutions</span></span>  
  
 <span data-ttu-id="babe6-104">Cette erreur se produit quand une classe générique implémente plusieurs paramétrages de la même interface générique et qu’il existe une substitution de paramètre de type qui rendrait les deux interfaces identiques.</span><span class="sxs-lookup"><span data-stu-id="babe6-104">This error occurs when a generic class implements more than one parameterization of the same generic interface, and there exists a type parameter substitution which would make the two interfaces identical.</span></span> <span data-ttu-id="babe6-105">Pour éviter cette erreur, implémentez une seule des interfaces ou modifiez les paramètres de type pour éviter le conflit.</span><span class="sxs-lookup"><span data-stu-id="babe6-105">To avoid this error, implement only one of the interfaces, or change the type parameters to avoid the conflict.</span></span>  
  
 <span data-ttu-id="babe6-106">L’exemple suivant génère l’erreur CS0695 :</span><span class="sxs-lookup"><span data-stu-id="babe6-106">The following sample generates CS0695:</span></span>  
  
```csharp

// compile with: /target:library

{  
}

{  
}  
```

## <a name="example"></a><span data-ttu-id="a11e1-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="a11e1-107">Example</span></span>  
 <span data-ttu-id="a11e1-108">L’exemple suivant génère l’erreur CS0754 :</span><span class="sxs-lookup"><span data-stu-id="a11e1-108">The following code generates CS0754:</span></span>

    public interface IF  
    {  
        void Part();  
    }

        }  
    }

 <span data-ttu-id="353ab-104">Une chaîne passée à une balise cref, par exemple, dans un \<exception > balise, référencé un membre qui n’est pas disponible dans l’environnement de génération en cours.</span><span class="sxs-lookup"><span data-stu-id="353ab-104">A string passed to a cref tag, for example, within an \<exception> tag, referred to a member that is not available within the current build environment.</span></span> <span data-ttu-id="353ab-105">La chaîne que vous passez à une balise cref doit correspondre au nom d’un membre ou d’un champ ayant une syntaxe correcte.</span><span class="sxs-lookup"><span data-stu-id="353ab-105">The string that you pass to a cref tag must be the syntactically correct name of a member or field.</span></span>

 <span data-ttu-id="353ab-107">L’exemple suivant génère l’erreur CS1574 :</span><span class="sxs-lookup"><span data-stu-id="353ab-107">The following sample generates CS1574:</span></span>  
  
```csharp  
// CS1574.cs  
// compile with: /W:1 /doc:x.xml  
using System;  
  
/// <exception cref="System.Console.WriteLin">An exception class.</exception>   // CS1574

// /// <exception cref="System.Console.WriteLine">An exception class.</exception>

{  
}  
  
class TestClass  
{  
   public static void Main()  
   {  
      try  
      {  
      }  
      catch(EClass)  
      {  
      }  
   }  
}  
```

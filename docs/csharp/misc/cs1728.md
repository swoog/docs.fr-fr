# <a name="compiler-error-cs1728"></a>Erreur du compilateur CS1728
Impossible de lier le délégué à 'membre', car il s’agit d’un membre de 'type'  
  
 Vous ne pouvez pas lier de délégués à des membres de types `Nullable` .  
  
## <a name="example"></a>Exemple  
 L’exemple suivant génère l’avertissement CS1728 :  
  
```csharp  
// CS1728.cs  
class Test  
{  
   delegate T GetT<T>();  
   delegate T GetT1<T>(T t);  
  
   delegate bool E(object o);  
   delegate int I();  
   delegate string S();  
  
   static void Main()  
   {  
      int? x = null;  
      int? y = 5;  
  
      GetT<int> d1 = x.GetValueOrDefault;   // CS1728  
      GetT<int> d2 = y.GetValueOrDefault;   // CS1728  
      GetT1<int> d3 = x.GetValueOrDefault;   // CS1728  
      GetT1<int> d4 = y.GetValueOrDefault;   // CS1728  
   }  
}  
```

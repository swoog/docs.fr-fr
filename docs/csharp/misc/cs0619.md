# <a name="compiler-error-cs0619"></a><span data-ttu-id="30dd8-102">Erreur du compilateur CS0619</span><span class="sxs-lookup"><span data-stu-id="30dd8-102">Compiler Error CS0619</span></span>
<span data-ttu-id="30dd8-103">'membre' est obsolète : 'texte'</span><span class="sxs-lookup"><span data-stu-id="30dd8-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="30dd8-104">Un membre de classe a été marqué avec la [obsolète] ((.. /.. / csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) d’attribut, tel qu’une erreur est émise lorsque le membre de classe est référencé.</span><span class="sxs-lookup"><span data-stu-id="30dd8-104">A class member was marked with the [Obsolete]((../../csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="30dd8-105">L’exemple suivant génère l’erreur CS0619 :</span><span class="sxs-lookup"><span data-stu-id="30dd8-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```

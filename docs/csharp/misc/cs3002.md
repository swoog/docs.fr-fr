# <a name="compiler-warning-level-1-cs3002"></a>Avertissement du compilateur (niveau 1) CS3002
Le type de retour de 'method' n’est pas conforme CLS  
  
 Une méthode [publique](../../csharp/language-reference/keywords/public.md), [protégée](../../csharp/language-reference/keywords/protected.md)ou `protected internal` doit retourner une valeur dont le type est conforme à la spécification CLS (Common Language Specification). Pour plus d’informations sur la conformité CLS, consultez [indépendance du langage et composants indépendants du langage](../../standard/language-independence.md).
  
## <a name="example"></a>Exemple  
 L’exemple suivant génère l’avertissement CS3002 :  
  
```csharp  
// CS3002.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```

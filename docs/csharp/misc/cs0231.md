# <a name="compiler-error-cs0231"></a><span data-ttu-id="50f5b-102">Erreur du compilateur CS0231</span><span class="sxs-lookup"><span data-stu-id="50f5b-102">Compiler Error CS0231</span></span>
<span data-ttu-id="50f5b-103">Un paramètre params doit être le dernier paramètre dans une liste de paramètres formels.</span><span class="sxs-lookup"><span data-stu-id="50f5b-103">A params parameter must be the last parameter in a formal parameter list.</span></span>  
  
 <span data-ttu-id="50f5b-104">Le paramètre [params](../../csharp/language-reference/keywords/params.md) prend en charge un nombre variable d’arguments et doit se trouver après tous les autres paramètres.</span><span class="sxs-lookup"><span data-stu-id="50f5b-104">The [params](../../csharp/language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="50f5b-105">Pour plus d’informations, consultez [Méthodes](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="50f5b-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="50f5b-106">L’exemple suivant génère l’erreur CS0231 :</span><span class="sxs-lookup"><span data-stu-id="50f5b-106">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs  
class Test  
{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}   
  
   static void Main()   
   {  
   }  
}  
```

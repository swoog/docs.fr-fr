# <a name="compiler-warning-level-3-cs1717"></a><span data-ttu-id="9d1ed-102">Avertissement du compilateur (niveau 3) CS1717</span><span class="sxs-lookup"><span data-stu-id="9d1ed-102">Compiler Warning (level 3) CS1717</span></span>
<span data-ttu-id="9d1ed-103">Assignation effectuée à la même variable ; souhaitiez-vous assigner un autre élément ?</span><span class="sxs-lookup"><span data-stu-id="9d1ed-103">Assignment made to same variable; did you mean to assign something else?</span></span>  
  
 <span data-ttu-id="9d1ed-104">Cet avertissement se produit quand vous assignez une variable à elle-même (par exemple `a = a`.</span><span class="sxs-lookup"><span data-stu-id="9d1ed-104">This warning occurs when you assign a variable to itself, such as `a = a`.</span></span>  
  
 <span data-ttu-id="9d1ed-105">Plusieurs erreurs courantes peuvent générer cet avertissement :</span><span class="sxs-lookup"><span data-stu-id="9d1ed-105">Several common mistakes can generate this warning:</span></span>  
  
- <span data-ttu-id="9d1ed-106">Écriture de `a = a` comme condition d’une instruction **if** , par exemple `if (a = a)`.</span><span class="sxs-lookup"><span data-stu-id="9d1ed-106">Writing `a = a` as the condition of an **if** statement, such as `if (a = a)`.</span></span> <span data-ttu-id="9d1ed-107">Vous souhaitiez probablement indiquer `if (a == a)`, qui est toujours vrai ; vous pouvez donc mentionner de façon plus concise `if (true)`.</span><span class="sxs-lookup"><span data-stu-id="9d1ed-107">You probably meant to say `if (a == a)`, which is always true, so you could write this more concisely as `if (true)`.</span></span>  
  
- <span data-ttu-id="9d1ed-108">Erreur de frappe.</span><span class="sxs-lookup"><span data-stu-id="9d1ed-108">Mistyping.</span></span> <span data-ttu-id="9d1ed-109">Vous souhaitiez probablement indiquer `a = b`.</span><span class="sxs-lookup"><span data-stu-id="9d1ed-109">You probably meant to say `a = b`.</span></span>  
  
- <span data-ttu-id="9d1ed-110">Dans un constructeur dans lequel le paramètre porte le même nom que le champ, sans utilisation du mot clé **this** : vous souhaitiez probablement indiquer `this.a = a`.</span><span class="sxs-lookup"><span data-stu-id="9d1ed-110">In a constructor where the parameter has the same name as the field, not using the **this** keyword: you probably meant to say `this.a = a`.</span></span>

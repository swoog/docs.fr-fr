 <span data-ttu-id="2b9d9-104">Vous avez tenté de substituer un type de données autre qu’une propriété en tant que [propriété](../../csharp/programming-guide/classes-and-structs/properties.md), ce qui n’est pas autorisé.</span><span class="sxs-lookup"><span data-stu-id="2b9d9-104">An attempt was made to override a nonproperty data type as a [property](../../csharp/programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>  
  
 <span data-ttu-id="2b9d9-105">L’exemple suivant génère l’erreur CS0544 :</span><span class="sxs-lookup"><span data-stu-id="2b9d9-105">The following sample generates CS0544:</span></span>  
  
```csharp

{  
   public int i;  
}

   public override int i {   // CS0544  
   // try the following line instead  
   // public new int i {

      {  
         return 0;  
      }  
   }  
}  
```

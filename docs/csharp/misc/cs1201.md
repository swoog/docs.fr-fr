# <a name="compiler-warning-level-1-cs1201"></a>Avertissement du compilateur (niveau 1) CS1201
La fonctionnalité 'fonctionnalité_incorrecte' est déconseillée. Utilisez 'fonctionnalité_correcte' à la place.  
  
 La fonctionnalité que vous essayez d’utiliser est désormais déconseillée. Mettez à jour votre code de manière à utiliser la fonctionnalité valide à la place.

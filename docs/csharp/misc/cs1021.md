# <a name="compiler-error-cs1021"></a><span data-ttu-id="4e78f-102">Erreur du compilateur CS1021</span><span class="sxs-lookup"><span data-stu-id="4e78f-102">Compiler Error CS1021</span></span>

<span data-ttu-id="4e78f-103">Constante intégrale trop grande</span><span class="sxs-lookup"><span data-stu-id="4e78f-103">Integral constant is too large</span></span>  
  
<span data-ttu-id="4e78f-104">Une valeur représentée par un littéral entier est supérieure à <xref:System.UInt64.MaxValue?displayProperty=nameWithType>.</span><span class="sxs-lookup"><span data-stu-id="4e78f-104">A value represented by an integer literal is greater than <xref:System.UInt64.MaxValue?displayProperty=nameWithType>.</span></span>  
  
<span data-ttu-id="4e78f-105">L’exemple suivant génère l’erreur CS1021 :</span><span class="sxs-lookup"><span data-stu-id="4e78f-105">The following sample generates CS1021:</span></span>  

```csharp
// CS1021.cs  
class Program
{
    static void Main(string[] args)
    {
        int a = 18_446_744_073_709_552_000;
    }
}  
```

<span data-ttu-id="4e78f-106">Le code suivant génère également l’erreur CS1021 :</span><span class="sxs-lookup"><span data-stu-id="4e78f-106">The following code also generates CS1021:</span></span>

```csharp
using System.Numerics;

class Program
{
    static void Main(string[] args)
    {
        var a = new BigInteger(18_446_744_073_709_552_000);
    }
}
```

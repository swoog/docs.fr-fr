# <a name="compiler-warning-level-2-cs0108"></a><span data-ttu-id="ec201-102">Avertissement du compilateur (niveau 2) CS0108</span><span class="sxs-lookup"><span data-stu-id="ec201-102">Compiler Warning (level 2) CS0108</span></span>

<span data-ttu-id="ec201-103">'member1' masque le membre hérité 'member2'.</span><span class="sxs-lookup"><span data-stu-id="ec201-103">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="ec201-104">Utilisez le mot clé new si le masquage est intentionnel.</span><span class="sxs-lookup"><span data-stu-id="ec201-104">Use the new keyword if hiding was intended.</span></span>  
  
 <span data-ttu-id="ec201-105">Une variable portant le même nom qu’une variable d’une classe de base a été déclarée.</span><span class="sxs-lookup"><span data-stu-id="ec201-105">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="ec201-106">Cependant, le mot clé [new](../../../csharp/language-reference/keywords/new.md) n’a pas été utilisé.</span><span class="sxs-lookup"><span data-stu-id="ec201-106">However, the [new](../../../csharp/language-reference/keywords/new.md) keyword was not used.</span></span> <span data-ttu-id="ec201-107">Cet avertissement vous informe que vous devez utiliser **new** ; la variable est déclarée comme si **new** avait été utilisé dans la déclaration.</span><span class="sxs-lookup"><span data-stu-id="ec201-107">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>  
  
 <span data-ttu-id="ec201-108">L’exemple suivant génère l’avertissement CS0108 :</span><span class="sxs-lookup"><span data-stu-id="ec201-108">The following sample generates CS0108:</span></span>  

```csharp
// CS0108.cs  
// compile with: /W:2  
using System;  
  
namespace x  
{  
   public class clx  
   {  
      public int i = 1;  
   }  
  
   public class cly : clx  
   {  
      public static int i = 2;   // CS0108, use the new keyword  
      // Use the following line instead:  
      // public static new int i = 2;  
  
      public static void Main()  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```

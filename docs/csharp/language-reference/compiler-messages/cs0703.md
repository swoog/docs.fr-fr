# <a name="compiler-error-cs0703"></a><span data-ttu-id="ab8de-102">Erreur du compilateur CS0703</span><span class="sxs-lookup"><span data-stu-id="ab8de-102">Compiler Error CS0703</span></span>
<span data-ttu-id="ab8de-103">Accessibilité incohérente : le type de contrainte 'identifier' est moins accessible que 'identifier'</span><span class="sxs-lookup"><span data-stu-id="ab8de-103">Inconsistent accessibility: constraint type 'identifier' is less accessible than 'identifier'</span></span>  
  
 <span data-ttu-id="ab8de-104">Une contrainte ne peut pas forcer le paramètre générique à être moins accessible que la classe générique elle-même.</span><span class="sxs-lookup"><span data-stu-id="ab8de-104">A constraint may not force the generic parameter to be less accessible than the generic class itself.</span></span> <span data-ttu-id="ab8de-105">Dans l’exemple suivant, alors que la classe générique C\<T> est déclarée public, la contrainte tente de forcer T à implémenter une interface interne.</span><span class="sxs-lookup"><span data-stu-id="ab8de-105">In the following example, while the generic class C\<T> is declared public, the constraint attempts to force T to implement an internal interface.</span></span> <span data-ttu-id="ab8de-106">Même si cela était autorisé, seuls les clients disposant d’un accès interne pourraient créer le paramètre pour la classe ; en conséquence, cette classe ne pourrait être utilisée que par les clients disposant d’un accès interne.</span><span class="sxs-lookup"><span data-stu-id="ab8de-106">Even if this were allowed, only clients with internal access would be able to create the parameter for the class, so in effect, the class could be used only by clients with internal access.</span></span>  
  
 <span data-ttu-id="ab8de-107">Pour supprimer cette erreur, vérifiez que le niveau d’accès de la classe générique n’est pas moins restrictif que les classes ou interfaces qui figurent dans les limites.</span><span class="sxs-lookup"><span data-stu-id="ab8de-107">To get rid of this error, make sure the access level of the generic class is not less restrictive than any classes or interfaces that appear in the bounds.</span></span>  
  
 <span data-ttu-id="ab8de-108">L’exemple suivant génère l’erreur CS0703 :</span><span class="sxs-lookup"><span data-stu-id="ab8de-108">The following sample generates CS0703:</span></span>  
  
```csharp  
// CS0703.cs  
internal interface I {}  
public class C<T> where T : I  // CS0703 – I is internal; C<T> is public  
{  
}  
```

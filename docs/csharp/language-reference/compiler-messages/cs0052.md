# <a name="compiler-error-cs0052"></a><span data-ttu-id="361db-102">Erreur du compilateur CS0052</span><span class="sxs-lookup"><span data-stu-id="361db-102">Compiler Error CS0052</span></span>
<span data-ttu-id="361db-103">Accessibilité incohérente : le type de champ 'type' est moins accessible que le champ 'field'</span><span class="sxs-lookup"><span data-stu-id="361db-103">Inconsistent accessibility: field type 'type' is less accessible than field 'field'</span></span>  
  
 <span data-ttu-id="361db-104">Le type d’un champ ne peut pas être moins accessible que le champ lui-même : en effet, toutes les constructions publiques doivent retourner un objet accessible publiquement.</span><span class="sxs-lookup"><span data-stu-id="361db-104">The type of a field cannot be less accessible than the field itself because all public constructs must return a publicly accessible object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="361db-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="361db-105">Example</span></span>  
 <span data-ttu-id="361db-106">L’exemple suivant génère l’erreur CS0052 :</span><span class="sxs-lookup"><span data-stu-id="361db-106">The following sample generates CS0052:</span></span>  
  
```csharp  
// CS0052.cs  
public class MyClass2  
{  
    // The following line causes an error because the field, M, is declared  
    // as public, but the type, MyClass, is private. Therefore the type is   
    // less accessible than the field.  
    public MyClass M;   // CS0052  
  
    private class MyClass  
    {  
    }  
    // One way to resolve the error is to change the accessibility of the type  
    // to public.   
    //public class MyClass  
    // Another solution is to change the accessibility of the field to private.  
    // private MyClass M;  
}  
  
    public class MainClass  
    {  
        public static void Main()  
        {  
        }  
    }  
```

## <a name="example"></a><span data-ttu-id="1cb43-110">Exemple</span><span class="sxs-lookup"><span data-stu-id="1cb43-110">Example</span></span>

<span data-ttu-id="1cb43-111">L’exemple suivant génère l’erreur CS1936 :</span><span class="sxs-lookup"><span data-stu-id="1cb43-111">The following example produces CS1936:</span></span>  

```csharp
// cs1936.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```

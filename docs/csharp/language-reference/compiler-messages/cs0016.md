# <a name="compiler-error-cs0016"></a>Erreur du compilateur CS0016
Impossible d’écrire dans le fichier de sortie 'file' — 'reason'  
  
 Le compilateur n’a pas pu écrire dans un fichier de sortie. Vérifiez le chemin d’accès au fichier pour vous assurer qu’il existe. Si un fichier généré précédemment se trouve déjà à cet emplacement, vérifiez qu’il est accessible en écriture et qu’aucun processus n’a placé de verrou sur le fichier. Par exemple, vérifiez que votre fichier exécutable ne s’exécute pas quand vous essayez d’effectuer une génération.

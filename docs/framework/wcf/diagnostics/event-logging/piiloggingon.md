# <a name="piiloggingon"></a>PiiLoggingOn
Id : 107  
  
 Gravité : Error  
  
 Catégorie : Traçage  
  
## <a name="description"></a>Description  
 Cet événement indique que la journalisation PII a été activée. Les informations sensibles seront enregistrées. Cet événement répertorie le nom et l'ID de processus.

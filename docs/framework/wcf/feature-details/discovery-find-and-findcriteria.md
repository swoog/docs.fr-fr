## <a name="findresponse"></a><span data-ttu-id="acba7-140">FindResponse</span><span class="sxs-lookup"><span data-stu-id="acba7-140">FindResponse</span></span>  
 <xref:System.ServiceModel.Discovery.FindResponse> <span data-ttu-id="acba7-141">a un <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> propriété de collection qui contient toutes les réponses envoyées en mettant en correspondance des services sur le réseau.</span><span class="sxs-lookup"><span data-stu-id="acba7-141">has an <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> collection property that contains any replies sent by matching services on the network.</span></span> <span data-ttu-id="acba7-142">Si aucun service n’a répondu, la collection est vide.</span><span class="sxs-lookup"><span data-stu-id="acba7-142">If no services replied, the collection is empty.</span></span> <span data-ttu-id="acba7-143">Si un ou plusieurs services ont répondu, chaque réponse est stockée dans un objet <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata>, qui contient l'adresse, le contrat et quelques informations supplémentaires sur le service.</span><span class="sxs-lookup"><span data-stu-id="acba7-143">If one or more services replied, each reply is stored in an <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> object, which contains the address, contract, and some additional information about the service.</span></span>  
  
 <span data-ttu-id="acba7-144">L'exemple suivant indique comment effectuer une opération de recherche dans le code.</span><span class="sxs-lookup"><span data-stu-id="acba7-144">The following example shows how to perform a find operation in code.</span></span>  
  
```  
// Create DiscoveryClient  
DiscoveryClient discoveryClient = new DiscoveryClient(new UdpDiscoveryEndpoint());  
  
// Create FindCriteria  
FindCriteria findCriteria = new FindCriteria(typeof(IPrinterService));  
findCriteria.Scopes.Add(new Uri("http://www.contoso.com/building1/floor1"));  
findCriteria.Duration = TimeSpan.FromSeconds(10);   
  
// Find ICalculatorService endpoints              
FindResponse findResponse = discoveryClient.Find(findCriteria);  
  
Console.WriteLine("Found {0} ICalculatorService endpoint(s).", findResponse.Endpoints.Count)  
```

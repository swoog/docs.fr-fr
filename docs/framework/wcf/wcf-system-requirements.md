# <a name="wcf-system-requirements"></a>Configuration requise WCF
Windows Communication Foundation (WCF) est une infrastructure de communication qui est utilisée pour créer des applications distribuées. La liste suivante décrit les exigences qui permettront à WCF à exécuter.  
  
## <a name="system-requirements"></a>Configuration système requise  
 WCF est installé par défaut sur [!INCLUDE[wv](../../../includes/wv-md.md)].  
  
 WCF peut également être installé sur [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], ou [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].  
  
 **Remarque** des fonctionnalités de WCF le Message Queuing (MSMQ) sont pris en charge uniquement sur [!INCLUDE[wv](../../../includes/wv-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)], et [!INCLUDE[wxppro](../../../includes/wxppro-md.md)].

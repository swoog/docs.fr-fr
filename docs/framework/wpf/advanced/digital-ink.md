# <a name="digital-ink"></a>Encre numérique
Cette section traite de l’utilisation de l’encre numérique dans le [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]. Traditionnellement trouvé uniquement dans le SDK Tablet PC, l’encre numérique est désormais disponible dans le noyau Windows Presentation Foundation. Cela signifie que vous pouvez désormais développer des applications Tablet PC à part entière à l’aide de la puissance de Windows Presentation Foundation.

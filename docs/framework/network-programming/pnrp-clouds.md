 <span data-ttu-id="c6960-108">Les clouds sont très liés aux interfaces réseau.</span><span class="sxs-lookup"><span data-stu-id="c6960-108">Clouds are tied very closely to network interfaces.</span></span>  <span data-ttu-id="c6960-109">Sur un ordinateur multirésident comprenant deux cartes réseau connectées à différents sous-réseaux, trois clouds sont retournés : un pour chacune des adresses link-local par interface, et un cloud global.</span><span class="sxs-lookup"><span data-stu-id="c6960-109">On a multi-homed machine with two network cards attached to different subnets, three clouds will be returned: one for each of the link local addresses per interface, and a single global scope cloud.</span></span>

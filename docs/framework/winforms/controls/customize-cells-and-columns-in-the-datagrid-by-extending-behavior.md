 <span data-ttu-id="3eb88-106">Vous pouvez créer une classe <xref:System.Windows.Forms.DataGridViewCell> personnalisée en dérivant de la classe de base <xref:System.Windows.Forms.DataGridViewCell> ou de l'une de ses classes dérivées.</span><span class="sxs-lookup"><span data-stu-id="3eb88-106">You create a custom <xref:System.Windows.Forms.DataGridViewCell> class by deriving from the <xref:System.Windows.Forms.DataGridViewCell> base class or one of its derived classes.</span></span> <span data-ttu-id="3eb88-107">Bien que vous puissiez afficher tout type de cellule dans tout type de colonne, vous créerez en général aussi une classe <xref:System.Windows.Forms.DataGridViewColumn> personnalisée spécialisée pour l'affichage de votre type de cellule.</span><span class="sxs-lookup"><span data-stu-id="3eb88-107">Although you can display any type of cell in any type of column, you will typically also create a custom <xref:System.Windows.Forms.DataGridViewColumn> class specialized for displaying your cell type.</span></span> <span data-ttu-id="3eb88-108">Les classes de colonne dérivent de <xref:System.Windows.Forms.DataGridViewColumn> ou de l'un de ses types dérivés.</span><span class="sxs-lookup"><span data-stu-id="3eb88-108">Column classes derive from <xref:System.Windows.Forms.DataGridViewColumn> or one of its derived types.</span></span>

# <a name="hscrollbar-and-vscrollbar-controls-windows-forms"></a>HScrollBar et VScrollBar, contrôles (Windows Forms)
Les contrôles de barre de défilement Windows Forms permettent une navigation facile au sein d'une longue liste d'éléments ou d'une grande quantité d'informations grâce au défilement horizontal ou vertical du contenu d'une application ou d'un contrôle. Les barres de défilement sont des éléments courants de l'interface Windows.

# <a name="iassemblynamefinalize-method"></a>IAssemblyName::Finalize, méthode
Cela permet [IAssemblyName](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-interface.md) objet pour libérer des ressources et effectuer d’autres opérations de nettoyage avant que son destructeur est appelé.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT Finalize ();  
```

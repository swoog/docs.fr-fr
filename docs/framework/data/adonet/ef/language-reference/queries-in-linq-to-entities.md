 <span data-ttu-id="0c562-113">Dans la requête, vous spécifiez exactement les informations que vous voulez récupérer à partir de la source de données.</span><span class="sxs-lookup"><span data-stu-id="0c562-113">In the query, you specify exactly the information that you want to retrieve from the data source.</span></span> <span data-ttu-id="0c562-114">Une requête peut également spécifier la manière dont ces informations doivent être triées, regroupées et mises en forme avant d'être retournées.</span><span class="sxs-lookup"><span data-stu-id="0c562-114">A query can also specify how that information should be sorted, grouped, and shaped before it is returned.</span></span> <span data-ttu-id="0c562-115">Dans LINQ, une requête est stockée dans une variable.</span><span class="sxs-lookup"><span data-stu-id="0c562-115">In LINQ, a query is stored in a variable.</span></span> <span data-ttu-id="0c562-116">Si la requête retourne une séquence de valeurs, la variable de requête doit elle-même être de type requêtable.</span><span class="sxs-lookup"><span data-stu-id="0c562-116">If the query returns a sequence of values, the query variable itself must be a queryable type.</span></span> <span data-ttu-id="0c562-117">Cette variable de requête n'effectue aucune action et ne retourne aucune donnée ; elle stocke uniquement les informations de requête.</span><span class="sxs-lookup"><span data-stu-id="0c562-117">This query variable takes no action and returns no data; it only stores the query information.</span></span> <span data-ttu-id="0c562-118">Une fois que vous avez créé une requête, vous devez l'exécuter pour extraire des données.</span><span class="sxs-lookup"><span data-stu-id="0c562-118">After you create a query you must execute that query to retrieve any data.</span></span>
